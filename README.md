# ELITE ARENA - Hub Performa Tinggi

ELITE ARENA adalah fasilitas pelatihan atletik premium yang dirancang untuk membantu atlet mencapai potensi maksimal mereka. Proyek ini merupakan situs web profil perusahaan yang modern, responsif, dan elegan.

## Fitur Utama

- **Desain Premium & Modern**: Menggunakan estetika profesional dengan tipografi yang kuat dan palet warna atletik.
- **Responsif Sepenuhnya**: Dioptimalkan untuk semua perangkat (PC, Tablet, dan Smartphone).
- **Animasi Halus**: Dilengkapi dengan efek _scroll-reveal_ dan transisi premium untuk pengalaman pengguna yang dinamis.
- **Navigasi Cerdas**: Header transparan yang berubah menjadi solid saat gulir (sticky header) dan menu mobile yang intuitif.
- **Konten Lokal**: Seluruh konten telah disesuaikan ke dalam Bahasa Indonesia yang profesional.

## Arsitektur Proyek

- `index.html`: Struktur semantik dengan praktek terbaik SEO.
- `index.css`: Sistem desain kustom menggunakan variabel CSS untuk pemeliharaan yang mudah.
- `script.js`: Logika interaksi untuk navigasi dan animasi performa tinggi.

## Cara Menjalankan

Cukup buka file `index.html` di browser pilihan Anda atau gunakan ekstensi Live Server untuk pengalaman pengembangan yang lebih baik.
